# MD005 - list-indent

Inconsistent indentation for list items at the same level.

**Tags:** bullet, ul, indentation

**Aliases:** list-indent

**Fixable:** Partial (ordered lists only)

## Rationale

List items at the same nesting level should start at the same column to maintain consistent visual structure and avoid confusing readers.

## Examples

### ❌ Incorrect (Unordered Lists)

```markdown
* Item 1
  * Nested item
* Item 2
   * Nested item (inconsistent indent - 3 spaces vs 2)
```

### ✓ Correct (Unordered Lists)

```markdown
* Item 1
  * Nested item
* Item 2
  * Nested item (consistent 2-space indent)
```

### ❌ Incorrect (Ordered Lists)

```markdown
1. First item
2.  Second item (extra space)
3. Third item
```

### ✓ Correct (Ordered Lists)

```markdown
1. First item
2. Second item
3. Third item
```

## Configuration

No configuration options for this rule.

## Auto-fix Behavior

**Ordered lists:** MD005 provides auto-fix for ordered list indentation inconsistencies.

**Unordered lists:** MD005 does NOT provide auto-fix for unordered lists. Use [MD007](md007.md) (ul-indent) instead, which handles unordered list indentation more precisely.

This is why MD005 is marked as "Partial" fixability.
